# EAT_Client
------------------
### Brief Introduction
> Eating AppointmenT(EAT)는 밥친구를 찾아주는 서비스를 제공합니다. 저마다의 다른 스케줄과 새로운 환경으로 인해 어쩔 수 없이 혼밥을 경험하게 됩니다. 이러한 상황을 해결하기 위해 프로젝트를 기획하게 되었습니다.


### Platform
- Android moblie


### Tool
- Android Studio


### Main Functions ( Client )
+ 로그인
+ 회원가입
  + 이메일 인증
+ 밥친구 찾기
  + 방 생성
  + 방 접속
    + 채팅
+ 피드백


### Using APIs
- Google Map API
- Gmail API
